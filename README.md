# AWS-serverless-microservice-powertuning
A serverless backend powered by AWS Lambda, API Gateway, and DynamoDB — scalable and low-maintenance. This project measures the performance and cost trade-offs of different memory configurations, validates real traffic behavior, and applies the optimal configuration.
